 Projeto Validação de Senha (Node.js)

Este projeto foi criado com o objetivo de treinar lógica de programação, modularização e validação de dados utilizando Node.js e JavaScript moderno (ES Modules).

Ele simula a criação de uma senha, validando regras básicas e solicitando ao usuário que tente novamente até que a senha seja considerada válida.

 Tecnologias utilizadas

Node.js

JavaScript (ES Modules)

prompt-sync (entrada de dados pelo terminal)

Como funciona

O usuário digita uma senha pelo terminal

A senha é enviada para a função de validação

Caso existam erros:

Todos os erros são exibidos

O usuário deve digitar a senha novamente

O processo se repete até a senha ser válida

Quando válida, o sistema exibe "Senha válida"
